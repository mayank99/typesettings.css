Darker. Bolder. Stronger.

& zero colors given!

Typesettings.css is inspired by traditional graphic design fundamentals and the perfect lightweight CSS framework for any minimal website or article design. No classes required.

How to use

  1. Include typesettings.css in the <head> section of your HTML: 

    <link rel="stylesheet" href="path/typesettings.css">

  2. Add data-typesettings attribute to the direct child element of the <body>: 

    <body>
      <div data-typesettings>
        …
      </div>
    </body>

Modifiers

Modifiers are additional options you can add to the data-typesettings attribute.

Example 

<div data-typesettings="sans-serif indented golden dark">
 … 
</div>
  • sans-serif

    data-typesettings="sans-serif" sets the body text to sans-serif and heading text to serif.

  • indented

    data-typesettings="indented" uses paragraph indents instead of paragraph spacing.

  • golden

    data-typesettings="golden" turns on the golden ratio between article width and white space and sets the optimal character count per line.

  • dark

    data-typesettings="dark" applies a dark theme to the article.

Bells & whistles

Page banner

If you want to enable a page banner (such as the banner on this very page), create a <header> tag with the attribute role="banner". 

<header role="banner">
 … 
</header>

Drop cap

You can activate drop cap by using a <p> inside the page banner. 

<header role="banner">
 <p>…</p> 
</header>
