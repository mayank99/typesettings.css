Darker. Bolder. Stronger.

& zero colors given!

Typesettings.css is inspired by traditional graphic design fundamentals and the perfect lightweight CSS framework for any minimal website or article design. No colors; no Javascript; no classes!

Basic Usage

  1. Include typesettings.css in the HTML file’s <head> section: 

    <head>
      <link rel="stylesheet" href="[path]/typesettings.css">
    </head>

    Note: typesettings.css is meant to be used alone, it is not compatible with any other frameworks. Also, basic resets are included, Normalize.css or Sanitize.css is not needed.

  2. Add data-typesettings attribute to the direct child element of the <body>: 

    <body>
      <div data-typesettings>
        …
      </div>
    </body>

Modifiers

Modifiers are additional options you can add to the data-typesettings attribute. Mix and match for the desired effects.

Example 

<div data-typesettings="sans-serif indented golden dark">
  …
</div>
  • sans-serif

    data-typesettings="sans-serif" sets the body text to sans-serif and heading text to serif.

  • indented

    data-typesettings="indented" uses paragraph indents instead of paragraph spacing.

  • golden

    data-typesettings="golden" turns on the golden ratio between article width and white space and sets the optimal character count per line.

  • dark

    data-typesettings="dark" applies a dark theme to the content.
